# Luas-Guide
- Sample Application to show Luas route and time

## Features Included
- Pull to Refresh
- Data binding
- MVVM Architecture
- RX Java for network call
- KOIN for dependency injection

## Project Structure
- Main Activity
  - LuasForecastFragment
    - LuasFragmentViewModel
      - LuasRepository
